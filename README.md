# Orthodox Church of Indonesia

We are one, holy, catholic, and apostolic Orthodox Church based on Indonesia under Gereja Orthodox Indonesia, hence the name straightforward means "Orthodox Church of Indonesia".

Under his Emminence Archbishop Kallinikos of Athens and All Greece, Archbishop Metropolitan Photios of Demetrias, and Bishop Daniel of Nikopolis and All Indonesia.

This project were intended for broadcasting Orthodoxy and Evangelical mission by **Parish St. Basil the Great** from Depok, West Java. Led by Fray (Paul) as his vision on expand the Orthodoxia Teaching into whole new modern world.

This website is an open-source [Next.js](https://nextjs.org) based project, which you could actually contribute.

## Getting Started

First, run the development server:

```bash
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
